# Hamster Run

A tiny hamster lives in your sidebar and runs on a wheel as you type. The faster you code, the faster it runs.

![Hamster Run Demo](images/demo.gif)

> Replace the image above with an actual capture (`images/demo.gif`).

---

## Features

- **Typing-driven wheel** — The hamster's wheel speed reacts to your typing speed in real time
- **Natural idle behavior** — When you stop typing, the hamster wanders left and right and occasionally sits down to rest
- **Feed it a snack** — Click anywhere in the panel to drop a sunflower seed; the hamster will walk over and eat it
- **Heart reward** — A heart pops above the hamster's head every time it finishes a snack
- **Hidden easter eggs** — Try typing at lightspeed, or recall a famous cheat code from the 80s

---

## Install

From the VSCode Marketplace:

```
ext install miiin.hamster-run
```

Or from the command line:

```bash
code --install-extension miiin.hamster-run
```

---

## Usage

1. Click the hamster icon in the activity bar to open the side panel
2. Just code as you normally would — the wheel reacts automatically to your typing
3. To feed the hamster, **click** anywhere inside the panel; a sunflower seed will drop at that spot
4. The hamster walks over, sits down to eat, then leaves a heart behind

---

## Settings

Configure via `settings.json`:

| Setting | Default | Description |
|---------|---------|-------------|
| `hamsterRun.idleThreshold` | `0.2` | Below this cps (characters per second), the hamster is considered idle |
| `hamsterRun.windowMs` | `2000` | Sliding window length (ms) used to compute typing speed |

## Known Limitations

- The hamster lives only inside the webview panel (it doesn't roam your editor)
- The activity bar icon is currently a PNG, so it may blend into some themes
